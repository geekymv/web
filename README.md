### Vue CLI 脚手架
```shell
1.全局安装
yarn global add @vue/cli 或 npm i @vue/cli -g

2.查看Vue版本
vue --version

3.创建项目
vue create project-name

4.启动项目
yarn serve 或 npm run serve

```

### Style 支持 less
```shell

<style lang="less">
</style>

1.给style 加上 lang="less"
2.安装依赖包 less less-loader

```
